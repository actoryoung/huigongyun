# 样例数据说明

本目录提供若干精选样例项目，用于说明“图纸驱动报价清单生成”课题的数据形态。

## 数据角色

每个项目下分为：

1. `输入资料`：参赛系统可读取和使用的正式输入。
2. `参考输出_报价结果`：由输入资料生成的报价结果样例，可作为参考答案、训练样例或评分基准。

报价结果不应作为正式测试输入。

## 样例项目

### 项目A_IDC机房

特点：

1. 包含 DWG 图纸。
2. 包含 PDF 系统图。
3. 包含参考报价结果。

适合用于基础图纸识别和报价清单导出说明。

### 项目B学校

特点：

1. 包含 DWG 询价图纸。
2. 包含 PDF 系统图。
3. 包含主元件清单。
4. 包含报价说明。
5. 包含参考报价结果。

适合用于多源资料融合、主元件清单解析和约束抽取。

### 项目C资料更新

特点：

1. 包含配置说明。
2. 包含主元器件清单。
3. 包含图纸版本对比文件。
4. 包含参考报价结果。

适合用于资料变更影响分析和长交期物料提示。

### 项目D中英文设备清单

特点：

1. 包含中英文混合设备清单。
2. 包含 DWG 图纸。
3. 包含 PDF 系统图。
4. 包含参考报价结果。

适合用于中英文字段理解、设备清单解析和物料归一。
